This part is meant for input type `range`.

## Example

To create a range slider, just use type `range` alongside the `neu-btn` classname.

<div class="element-renderer">
  <input class="neu-input" type="range" min="0" max="10" value="3" />
</div>

```html
<input class="neu-input" type="range" min="0" max="10" value="3" />
```
